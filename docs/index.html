🔌 Wallbox Gateway — Web Installer

Local BLE → MQTT bridge for Wallbox Pulsar chargers. No cloud, full Home Assistant control.

One-click install

⚠️ Your browser doesn't support Web Serial — use desktop Chrome or Edge. ⚠️ This page must be served over HTTPS to flash.

Flashes Wallbox Gateway v3.0.1 (bootloader, partitions, firmware) with a full erase. Takes about a minute. ESP32-S3 boards only — original ESP32 / ESP32-C3 are not supported.

Needs desktop Chrome or Edge (Web Serial). Safari and Firefox can't flash — see the manual install guide.

Before you click

  1. Plug the ESP32-S3 into this computer with a USB-C data cable (not charge-only).
  2. Click Install and pick the serial port that appears.
  3. When flashing finishes, the gateway starts a Wallbox-Setup Wi-Fi hotspot — join it to configure Wi-Fi, MQTT and your charger.

After the first flash, all future updates are over-the-air from the gateway's own web UI — no cable needed.
